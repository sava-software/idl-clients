# Fuzz seed corpora

Seed provenance lives here — next to, never inside, a corpus directory, where a
file would itself be fed to the harness as a seed. The replay test that runs
every seed inside `check` is generated by the sava-build hardening plugin
(`StakePoolStateFuzzSeedReplayTest`); the harness contract is documented on
`StakePoolStateFuzz`.

## `stakePoolState/`

- `jito` — real mainnet Jito stake-pool account (611 bytes), fetched
  2026-07-16. Exercises the `Option`-steered offsets and `FutureEpochFee`
  branches that from-scratch inputs rarely assemble; the same account backs the
  `StakePoolStateTests` fixtures.

`validatorList` and `precompileOffsets` are deliberately unseeded: every input
prefix is already valid, so the whole space is reachable from scratch.
